# Infobiotics Workbench

## Requirements
* [Vagrant](https://www.vagrantup.com/downloads.html)
* [Virtualbox](https://www.virtualbox.org/wiki/Downloads) 4.0 and up

## User Installation
`$ cd vagrant_user`  
`$ vagrant up`

## Developer Installation
`$ cd vagrant_developer`  
`$ vagrant up`

## Usage
`$ cd vagrant_[role]`  
`$ vagrant up`
